Display tipai

Display tipas tai CSS savybės reikšmė. Yra 3 primityvūs display tipai: block, inline ir inline-block. CSS savybė display pasako kaip bus atvaizduojamas html elemntas.

Kai kuriems elementams naršyklė suteikia šiuos display tipus automatiškai.

Inline

Tai pagrindinis display tipas taikomas visiems vartotojo sukurtiems elementams. Jeigu sukuriame savo html elementą, jo tipas bus inline. Šis tipas pasako, kad elementas bus suvokiamas kaip tekstas. Elementai turintys šį tipą negali turėti aukščio ir pločios nustatymų. Jeigu nustatysite aukščio ar pločio nustatymus, jie paprasčiausiai neveiks.

Taip pat šiems elementams nekorektiškai veikia ir box-model savybės. Paraščių(margin) savybės veikia tik į šonus, o kamšalo(padding) saybės pritaikomos fonui, tačiau neturi įtakos elemnto kontūrų padidinimui.

Šį display tipą, turėtumėte naudoti tekstui, ikonoms arba nuotraukoms.

Tai vienintelis display tipas, kuris nėra keturkampis. Šio tipo elementai(tekstas) gali nusikelti per dvi ar daugiau eilučių;

Naršyklės turi rekomenduojamus elementus, kuriems nustato šį tipą. Dažniausiai naudojami yra:

  • <span>
  • <a>
  • <strong>
  • <em>

Inline-block

Tai display tipas, kuris atvaizduoja elementus rašymo tvarka: iš kairės į dešinę (taip pat kaip ir inline). Priešingai negu inline tipas display: inline-block elementai gali turi dimensijų nustatymus, bei box-model savybes. Kadangi tai yra blokai jie negali būti keliose eilutės, t.y. Jeigu elementas pilnai netelpa į tėvinio elemento plotį, jis perkeliamas į sekančią eilutę.

Naršyklės turi rekomenduojamus elementus, kuriems nustato šį tipą. Dažniausiai naudojami yra:

  • <input>
  • <select>
  • <button>
  • <text-area>

Pavyzdžiai

Iš kairės į dešinę atvaizduojami blokai, kurie nusikelia į kitą eilutę

Block

Tai atvaizdavimo tipas display: block, kuris užsirezervuoja visą tėvinio elemento plotį. To pasekoje rašant display: block elementus vienas po kitos, jie atvaizduojami stulpeliu - lyg namo aukštai. Net jeigu elementui nurodome, kad jis užims ne pilną tėvinio elemento plotį, kiti elementai nepateks į užrezervuotą vietą. Būtent todėl šie elementai ir atsivaizduoja venas po kitu.

Šiems elementams galima nustatyti plotį(mažesnį nei tėvinio elemento) ir tuomet naudoti margin: auto savybę atsistumti nuo kairės pusės, dešnės pusės arba abiejų pusių vienu metu. Kuomet nustatote atstūmimą (margin: auto) tiek iš kairės, tiek iš dešinės pusės - elmentas atvaizduojamas per vidurį tėvinio elemento.

Naršyklės turi rekomenduojamus elementus, kuriems nustato šį tipą. Dažniausiai naudojami yra:

  • <div>
  • <header>
  • <nav>
  • <section>
  • <footer>
  • <main>
  • <article>
  • <h1> - <h6>
