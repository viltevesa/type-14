CSS pagrindai

CSS tai kompiuterinė kalba skirta stilizuoti HTML elementus pagal pasirinktą selektorių.

Selektoriai gali būti primityvūs ir kompleksiniai. Dažniausiai kompleksiniai selektoriai yra naudojami kuomet reikia padidinti selektoriaus prioritetą.

Aprašant CSS selektorius yra atvejų kuomet tą patį elementą pasirenka keli selektoriai. Beveik visada įvyksta stiliaus konfliktai - skirtingi selektoriai nustato tas pačias savybes elementui. Tam yra skirti CSS prioritetai, kurie nusako kuris selektorius turės didesnę įtaką konkuruojant su kitais CSS selektoriais.

Primityvūs selektoriai

CSS selektorius tai kriterijus pagal ką pasirinkti elementus kuriuos stilizuosime.

Populiariausi CSS selektoriai yra tokie:

  • pagal elemento tipą
    • html{...} - stilizuojami visi elementai kurių tag name yra html
    • body{...} - stilizuojami visi elementai kurių tag name yra body
    • div{...} - stilizuojami visi elementai kurių tag name yra div
  • pagal klasę
    • .button{...} - stilizuojami visi elementai kurie turi attributą class ir jame yra įrašyta klasė button
    • .card{...} - stilizuojami visi elementai kurie turi attributą class ir jame yra įrašyta klasė card
    • .pt-3{...} - stilizuojami visi elementai kurie turi attributą class ir jame yra įrašyta klasė pt-3
  • pagal atributą
    • [href]{...} - stilizuojami visi elementai kurie turi attributą href
    • [id]{...} - stilizuojami visi elementai kurie turi attributą id
    • [type=text]{...} - stilizuojami visi elementai kurie turi attributą type ir to atributo reikšmė yra text
  • pagal id
    • #navbar{...} - stilizuojami visi elementai kurie turi attributą id ir jo reikšmė yra navbar
    • #main-link{...} - stilizuojami visi elementai kurie turi attributą id ir jo reikšmė yra main-link
    • #footer{...} - stilizuojami visi elementai kurie turi attributą id ir jo reikšmė yra footer

Pavyzdžiai

Pasirenkami elementai kurie turi attributą class ir jame įrašytas klasės pavadinimas card, pvz.:

<div class="card">...</div>
<article class="card">...</article>
<article class="card d-flex">...</article>

Pasirenkami elementai kurie turi attributą type ir jo reikšmė yra text, pvz.:

<input type="text" />
<input type="text" class="form-control" />
<input type="text" name="city" />

CSS prioritetai

CSS prioritetai tai taisyklės, kurios nusako selektoriaus stiprumą lyginant su kitais selektoriais.

Prioritetų grupės

Yra 3 prioritetų grupės. Jeigu CSS selektorius turi aukštesnį 1. grupės prioritą, tuomet antras nėra tikrinamas. Kitaip tariant, 2. grupės prioritetas tikrinamas tik tuomet, kuomet 1. grupės prioritetai sutampa.
3. grupės prioritetas tikrinamas tik tuomet, jeigu 1. ir 2. grupių prioritetai sutampa (yra vienodi).

2. Selektoriaus apibūdinimas

CSS selektoriaus apibūdinimas tai papildomų kriterijų aprašymas. Dažniausiai naudojamas tam, kad suteikti aukštesnį prieoritetą perrašant stilius.

Selektorius div.square turės aukštesnį prioritetą nei .square. Todėl elemento su klase .square išmatavimai bus 200x200.

Selektoriaus apibūdinimo(specificity) stiprumą ir grupes galite peržvelgti užvedus pelytę ant selektoriaus, taip kaip iliustracijoje dešinėje.

3. Selektoriaus eiliškumas

Kuomet selektorių 1. ir 2. grupių prioritetai sutampa, tuomet aukštesnį prioritetą turės tas selektorius, todėl šiuo kuris aprašytas vėliau. Todėl šiuo atveju aprašyto elemento fono spalva bus mėlyna.

Svarbi paminėti, kad failų įtraukimo tvarka irgi turi įtaką. Vėliau įtraukiami failai, turi aukštesnį prioritetą. Būtent todėl bibliotekų ir šriftų failus visuomet turite įtrauksi aukščiau nei savo kurtus failus. Įtraukiant failus tokia tvarka jūsų failai įgaus aukštesnį prioritetą.
