Position

Standarkiškai(default), jeigu nepasakome kitaip, visi elementai turi nustatytą CSS savybę position: static. Tai reiškia kad elementai yra nekintačios tėkmės (static flow). Static flow - tai elementų pozicionavimas iš kairės į dešinę, o kuomet elementai nebetelpa į tėvinio elemento plotį, perkeliami į sekančią eilutę.

Yra ir kitų posicionavimo tipų: relative, fixed, absolute ir sticky. Šie pozicionavimo tipai gali atsispirti nuo kitų elementų arba nuo jų nekintančios (static) vietos. Tai daroma naudojant savybes: top, right, bottom ir left. Static yra vienintelis pozicionavimo tipas, kuriam neturi įtakos savybės top, right, bottom ir left. Todėl jis ir yra vadinamis static.

Static

Static elementai nepriklauso nuo saybių top, right, bottom ir left ir yra atvaizduojami pagal nekintančią tėkmę(static flow).

Static flow - tai elementų pozicionavimas iš kairės į dešinę, o kuomet elementai nebetelpa į tėvinio elemento plotį, perkeliami į sekančią eilutę.

Fixed

Tai toks pozicionavimo būdas kuomet naudodami savybes top, right, bottom ir left gali 'atsispirti' nuo matomos turinio dalies (viewport). Užstančius elementui position: fixed jis dingsta iš static flow. Greta esantys elementai jo nemato, taip pat ir tėvinis elementas. position: fixed elementai išsižada savo tėvų, brolių ir sesių ir atsispira nuo viewport.

Absolute

Postion absolute naudoja savybes top, right, bottom ir left kad atsipirti nuo pirmo tėvinės hierarchijos elemento, kurio CSS savybė position NĖRA static.. Atsipirdamas nuo surasto NE STATIC elemento, position: absolute elementas dingsta iš static flow, todėl kiti elementai jo nebemato.

Dažniausiai praktikoje naudojame position: relative kad sukurti elementui atspirties tašką. Taip daroma todėl, kad jeigu position: relative elementui nenurodote poslinkio/atspirties savybių jis nepakinta ir elgiasi kaip position: static

Jeigu elementas position: absolute neranda atspirties taško (tėvinės hierarchijos elemento su position - NE STATIC), tuomet atsispiria nuo puslapio viršaus viewport kontūrų. Kitaip tariant, atsispirmia nuo šakninio puslapio elemento <html>
