Box Model

Box model tai kiekvieno HTML elemento struktūrizavimas sekcijomis/stačiakampiais. Kiekvienas HTML elementas (išskyrus per kelias eilutes persikeliantys display: inline elementai) turi 4 sekcijas: content, padding, border ir margin. Margin, padding ir border sekcijos gali turėti atskirus nustatymus į visas (keturias) puses. Content(turinio) išmatavimai standartiškai nustatomi naudojant width ir height savybes (jeigu box-sizing savybė nebuvo pakeista).

Standartiškai height ir width savybės nustatomos content sekcijai. Tačiau tai galima pakeisti naudojant savybę box-sizing.

Box model sekcijos

  • content - turinys
  • padding - kamšalas
  • border - sienelės
  • margin - paraštės

Padding ir margin CSS sayvybės rašymo trumpiniai

Kuomet rašome margin ir padding savybes, galime jas aprašyti atskirai į kiekvieną pusę, arba naudojant trumpuosius užrašymus.

Pavyzdžiai

  • 1 dydis - į visas puses
    margin: 10px;
    • margin-top: 10px
    • margin-right: 10px
    • margin-bottom: 10px
    • margin-left: 10px
  • 2 dydžiai - viršus ir šonai
    margin: 10px 20px;
    • margin-top: 10px
    • margin-right: 20px
    • margin-bottom: 10px
    • margin-left: 20px
  • 3 dydžiai - viršus, šonai ir apačia
    margin: 10px 20px 30px;
    • margin-top: 10px
    • margin-right: 20px
    • margin-bottom: 30px
    • margin-left: 20px
  • 4 dydžiai - viršus, dešinė, apačia ir kairė
    margin: 10px 20px 30px 40px;
    • margin-top: 10px
    • margin-right: 20px
    • margin-bottom: 30px
    • margin-left: 40px

Box sizing

Tai CSS savybė kuri pasako, pagal kokią box-model kontūrų dalį bus suvokiamas elementų eukštis ir plotis. Standartinė savybė CSS savybės box-sizing reikšmė yra content-box. Kuomet nustatyta ši reikšmė padding'as pridedas papildomai. Pavyzdžiui, jeigu elemento išmatavimai yra 40x40 ir papildomai pridedama padding'o, tuomet vizualūs išmatavimai bus 60x60.

Kuomet formuojame puslapio išdėstymą pagal parengtus dizainus, daug patogiau naudoti kitą šios savybęs reikšmę - box-sizing: border-box. Nustačius šią reikšmę, aukštis ir plotis suvokiamas pagal HTML elemento sienelę, o padding'as pridemas į elemento visų. Tokiu atveju pridėjus padding'o vizualūs išmatavimai nesikeičia. Paddingas tiesiog mašina turinio dalį.
