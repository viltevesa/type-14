Bootstrap intro

Bootstrap tai CSS savybių ir JavaScript kodo rinkinys, leidžiantis kurti turinį nerašant savo stiliaus. Visi stiliaus komponentai pasižymi vienoda stilistika:
Pagrindiai stiliai aprašomi komponenete, pvz.:.btn, o to komponento variacijos įgalinamos papildomomis klasėmis, kurios prasideda taip pat kaip komponentas, pvz: .btn-primary.

Taip pat bootstrap turi pagalbinių klasių rinkinius - be variacijų. Šios klasės skirtos išdėstymui, tarpams, spalvoms ir t.t. . pvz.: text-center, bg-secondary, d-flex ir t.t..

Kadangi šiuo metu puslapiai dažniau yra atidaromi su mobiliaisiais irenginiais ne stacionariais kompiuteriai Bootstrap suteikia papildomas variancijas visoms klasės naudojant ekrano dydžius: sm, md, lg, xl, xxl

Skirtingiems komponentų išdėstymams patogu naudoti specialų Bootstrap grid. Kuris yra grįstas display: flex ir flex savybių pagrindu.

Bootstrap spalvos

Bootstrap turi 8 pagrindines spalvas:

  • primary
  • secondary
  • success
  • warning
  • danger
  • info
  • light
  • dark

Šios spalvos gali būti taikomos kaip variacijos daugumai Bootstrap komponentų ir pagalbinėms klasėms

Grid

Bootstrap grid yra grįstas display: flex ir flex savybių pagrindu. Jis naudoja 12 dalių sistemą. Kiekvieną eilutė .row turi 12 menamų dalių, kurias gali užimti stulpeliai .col-*.
Stulpeliai bandomi talpinti į vieną eilutę, tačiau susidarius "spūsčiai" (netelpant elementams) jie yra perkeliami į sekančia eilutę. Tokiu būdu Bootstrap grid gali būti naudojamas ir dviejų dimensijų išdestymams.

Stulpelių klasės (col-*) gali turėti lužio taškų variacijas. Tokiu būdų išgaunami dizainai pritaikyti visiems ekrano dydžiams.
