Flex

Tai atvaizdavimo tipas, kuris yra elementą veikia kaip display: block, tačiau papildomai gali valdyti pirmo lygio(tiesioginių) vaikų išdėstymą. Elementai, kurie turi tiesioginį tėvą su display: flex gali papildomai koreguoti savo išdėstymo nustatymus specialiomis flex-child CSS savybėmis. Daugiau apie kiekvieną savybę galite paskaityti šiame straipsnyje:
Kairėje pusėje išvardintos savybės, kurias galite taikyti tėviniam elementui - tam kuriam užstatėte display: flex
Dešinėje pusėje išvardintos savybės, kurias galite taikyti vaikiniam elementui - turinčiam tiesioginį tėvą su CSS savybe display: flex
